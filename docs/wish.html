Interactive Tk console (wish)

This runs the real mini-tcl.lua + canvas.lua + tk.lua under fengari (a Lua VM in JavaScript). Type authentic Tk commands below — one per line — and the widgets appear and update on the canvas. Click and type on the canvas to drive them (buttons, the entry, the checkbutton, the scale).













Try this sequence, one line at a time:


button .b -text "Click me" -command {incr n}
label .out -textvariable n
set n 0
pack .b -side top -pady 8
pack .out -side top
entry .e -textvariable who -width 16
pack .e -side top -pady 8
scale .s -from 0 -to 100 -variable v -length 400
pack .s -side top


Then click the button (the count rises), click the entry and
type, and drag the slider — all on the canvas.
